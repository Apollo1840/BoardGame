# 基姆大陆(Gemu Land)

拥有一张手牌 **野猪战士**，对方场上有一张 **象灵**。

每到我方回合，抽两张牌，抽到 **麒麟附体** 和 **神兽守护**。

召唤 **野猪战士** 使用 **麒麟附体**， 攻击对方**象灵**（对方信仰-5）。覆盖**神兽守护**回合结束。

对方召唤 **犀牛战车** 攻击 **野猪战士**。 触发我方 **神兽守护**。对方覆盖一张怪物卡。

我方抽两张牌，抽到 **雪女.玖** 和 **对抗勋章**。

我方使用 灵力，召唤 **雪女.玖**，使用预言卡**对抗勋章**， 然后使用**雪女.玖**效果破坏对方覆盖一张怪物卡。**野猪战士** 与 **雪女.玖** 一同攻击。
